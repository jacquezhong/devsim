## 四、与论文的差异分析

### 4.1 实现层面的差异

| 方面 | 论文 | 我们的仿真 | 影响 |

### 4.2 结果层面的差异

| 指标 | 论文 | 我们的结果 | 分析 |
|------|------|-----------|------|
| **暗电流数量级** | ~10^-2 A/cm2 | ~10^-17 A/cm2 | **我们的结果过低** |
| **Rule 07对比** | 低于限值 | 远低于限值 | 趋势一致，但程度不同 |
| **温度依赖** | 未详细研究 | 固定100K | 需进一步研究 |

### 4.3 暗电流差异的可能原因

**为什么我们的暗电流比论文低很多？**

1. **理想化模型**
   - 无缺陷辅助隧穿
   - 无表面/界面态复合
   - 无位错/陷阱

2. **参数设置**
   - Auger系数可能偏小
   - 载流子寿命可能偏长
   - 迁移率模型简化

3. **物理机制缺失**
   - 带带隧穿（Band-to-band tunneling）
   - 缺陷辅助隧穿（TAT）
   - 表面漏电流

4. **计算设置**
   - 理想欧姆接触（无接触电阻）
   - 完美界面（无界面势垒）

**但核心结论一致：**
- ✅ 器件设计优于Rule 07
- ✅ 零VBO设计有效

### 8.3 关键公式实现

**泊松方程：**
```
∇·(ε∇φ) = -q(p - n + Nd)
```

**电子电流（Scharfetter-Gummel）：**
```
Jn = q·μn·V_t·(n1·B(Δφ/V_t) - n0·B(-Δφ/V_t))/Δx
```

**SRH复合：**
```
USRH = (n·p - ni²) / (τp(n+ni) + τn(p+ni))
```

**Auger复合：**
```
UAuger = (n·p - ni²)·(Cn·n + Cp·p)
```
