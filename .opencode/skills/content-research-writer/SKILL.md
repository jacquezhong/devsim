# Content Research Writer Skill

This skill acts as your writing partner, helping you research, outline, draft, and refine content while maintaining your unique voice and style.

## When to Use This Skill

Use this skill when:
- Writing blog posts, articles, or newsletters
- Creating educational content
- Drafting research-based content
- Needing help with citations and references
- Iterating on content structure and flow
- Improving engagement and readability

## Core Workflow

### Phase 1: Research & Planning

**Understand the Topic**
1. Clarify the main message or thesis
2. Identify target audience
3. Define key questions to answer
4. Determine required depth and scope

**Conduct Research**
- Search for relevant sources
- Identify authoritative references
- Extract key facts and data
- Find supporting evidence
- Note contradictory viewpoints

**Create Outline**
- Draft main sections
- Organize logical flow
- Allocate space to each section
- Identify key points per section
- Plan transitions

### Phase 2: Drafting

**Section-by-Section Writing**

For each section:
1. **Write the hook** - Grab attention immediately
2. **Present key points** - Clear, supported arguments
3. **Add evidence** - Facts, data, examples, quotes
4. **Include citations** - Proper attribution
5. **Craft transitions** - Smooth flow between ideas

**Writing Principles**
- Start with the most important information
- Use active voice
- Vary sentence length
- Include concrete examples
- Address reader questions proactively

### Phase 3: Refinement

**Real-Time Feedback Loop**

After each section:
1. Review for clarity
2. Check argument strength
3. Verify citations
4. Assess engagement
5. Suggest improvements

**Iterative Improvement**
- Revise based on feedback
- Strengthen weak points
- Add missing context
- Improve flow
- Polish language

## Key Features

### Research Integration

**Finding Sources**
- Identify credible references
- Extract relevant information
- Synthesize multiple viewpoints
- Maintain source diversity

**Citation Management**
- Add inline citations
- Create reference lists
- Follow citation styles (APA, MLA, Chicago, etc.)
- Ensure attribution accuracy

### Hook Improvement

**Types of Hooks**
- **Question**: Pose thought-provoking questions
- **Statistic**: Start with surprising data
- **Story**: Use narrative opening
- **Quote**: Lead with impactful statement
- **Problem**: Present a challenge to solve

**Hook Optimization**
- Test multiple openings
- Measure engagement potential
- Ensure relevance to content
- Create curiosity

### Outline Iteration

**Structural Options**
- Chronological
- Problem-Solution
- Compare-Contrast
- Cause-Effect
- Topical

**Outline Refinement**
- Reorganize for flow
- Balance section lengths
- Strengthen logical connections
- Remove redundancy

## Section-by-Section Guidance

### Introduction

**Components**
1. Hook (1-2 sentences)
2. Context/Background (1-2 paragraphs)
3. Main thesis/argument (1 sentence)
4. Roadmap (optional, 1 paragraph)

**Best Practices**
- Start strong
- Provide necessary context
- State purpose clearly
- Preview structure

### Body Sections

**Structure Each Section**
1. Topic sentence
2. Supporting evidence
3. Analysis/explanation
4. Transition to next section

**Guidelines**
- One main idea per paragraph
- Use evidence effectively
- Explain significance
- Maintain logical flow

### Conclusion

**Elements**
1. Restate main points
2. Synthesize key insights
3. Call to action or implication
4. Memorable closing

**Avoid**
- Introducing new information
- Simply summarizing
- Being too vague
- Weak endings

## Writing Quality Standards

### Clarity
- Use precise language
- Define technical terms
- Avoid ambiguity
- Be concise

### Engagement
- Write for your audience
- Use active voice
- Include examples
- Ask rhetorical questions

### Credibility
- Cite sources properly
- Present balanced views
- Acknowledge limitations
- Use qualified statements

### Flow
- Logical progression
- Smooth transitions
- Consistent tone
- Appropriate pacing

## Iteration Process

1. **Draft**: Write initial version
2. **Review**: Analyze against standards
3. **Feedback**: Identify improvement areas
4. **Revise**: Make targeted changes
5. **Repeat**: Continue until satisfied

## Collaboration Approach

**Maintaining Your Voice**
- Preserve your unique style
- Respect your perspective
- Enhance, don't replace
- Follow your preferences

**Partnership Dynamic**
- Suggest, don't dictate
- Explain reasoning
- Offer options
- Support decisions

## Best Practices

**Planning**
- Invest time in research
- Create detailed outlines
- Know your audience
- Define clear objectives

**Writing**
- Write freely first, edit later
- Focus on one section at a time
- Support all claims
- Maintain consistency

**Revising**
- Take breaks between drafts
- Read aloud
- Get external feedback
- Check all requirements

**Publishing**
- Proofread carefully
- Verify all links/citations
- Test readability
- Format properly
